# Pantalla de Carga Simple

Este es un script sencillo de pantalla de carga que ocupa toda la pantalla y muestra el mensaje "El script se está cargando".

## Características
- Ocupa el 100% del ancho y alto de la ventana.
- Incluye una animación de carga (spinner).
- El mensaje es personalizable.
- Se oculta automáticamente cuando la página termina de cargar (con un retraso simulado de 3 segundos).

## Cómo usar
Simplemente abre el archivo `index.html` en cualquier navegador.
